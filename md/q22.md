# 22. 括号生成
数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

示例：

输入：n = 3
输出：[
       "((()))",
       "(()())",
       "(())()",
       "()(())",
       "()()()"
     ]

# 用例
```
3
0
1
```

# 题解

当剩余放置的left小于right时，才能放右括号，这样一定是匹配的

(() 符合

) 不符合

## 回溯

定义辅助的递归函数

当left>0时，放左

递归退出来后，记得擦除当前的括号

当right>left时，放右

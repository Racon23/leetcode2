# 144. 二叉树的前序遍历
给定一个二叉树，返回它的 前序 遍历。

 示例:

输入: [1,null,2,3]  
   1
    \
     2
    /
   3 

输出: [1,2,3]
进阶: 递归算法很简单，你可以通过迭代算法完成吗？


# 用例
```
[1,null,2,3]
[]
[1]
[1,2,3,4,5,6,7]
[1,2,null,null,3,4]
```

# 题解

中左右

## moris魔改

1. 处理根节点，找左节点的最右子节点，连接到根，移到左
2. 左空，处理，移到右
3. 右空，退出

区别

前序遍历要在连接时就处理根

然后才处理 左

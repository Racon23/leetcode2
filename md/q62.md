# 62. 不同路径
一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

![](./q62_1.png)

示例 1：

输入：m = 3, n = 7
输出：28

示例 2：

输入：m = 3, n = 2
输出：3

解释：
从左上角开始，总共有 3 条路径可以到达右下角。
1. 向右 -> 向右 -> 向下
2. 向右 -> 向下 -> 向右
3. 向下 -> 向右 -> 向右
示例 3：

输入：m = 7, n = 3
输出：28
示例 4：

输入：m = 3, n = 3
输出：6
 
提示：

1 <= m, n <= 100
题目数据保证答案小于等于 2 * 109

# 题解

## 回溯 必定爆炸

## 逆序倒推

rem[i][j] = rem[i+1][j]+rem[i][j+1]

## 组合

从左上角移动到右下角，要走m+n-2步，其中往右走m-1步

即从m+n-2中有m-1选择的组合数

ans = C(m+n-2,m-1) 底数是m+n-2
    = A(m+n-2,m-1) / (m-1)!
    = m+n-2 m+n-3 ... m+n-2- m-1 + 1 / m-1 !
    = m+n-2 ... n / m-1 !

    = (m+n-2)!/ (m-1)!*(m+n-2- m-1)!
    = (m+n-2)! / (m-1)!* (n-1)!

A(n,m) = n n-1 ... n-m+1

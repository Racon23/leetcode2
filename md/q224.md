# 224. 基本计算器
实现一个基本的计算器来计算一个简单的字符串表达式 s 的值。

示例 1：

输入：s = "1 + 1"
输出：2
示例 2：

输入：s = " 2-1 + 2 "
输出：3
示例 3：

输入：s = "(1+(4+5+2)-3)+(6+8)"
输出：23
 

提示：

1 <= s.length <= 3 * 105
s 由数字、'+'、'-'、'('、')'、和 ' ' 组成
s 表示一个有效的表达式

# 用例
```
""1 + 1"
" 2-1 + 2 "
"(1+(4+5+2)-3)+(6+8)"
"-2+1"
"+1-2"
"1-(4+5)""
```

# 题解

## 数字栈和符号栈

- `(`入符号栈
- `1-9`入数字栈
- `+ -`入符号栈
- ` `无视
- `)`弹出`(`

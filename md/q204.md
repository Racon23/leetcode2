# 204. 计数质数
统计所有小于非负整数 n 的质数的数量。

示例 1：

输入：n = 10
输出：4
解释：小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。
示例 2：

输入：n = 0
输出：0
示例 3：

输入：n = 1
输出：0
 

提示：

0 <= n <= 5 * 106

# 用例
```
10
0
1
```

# 题解

## 枚举

i * i<= x 

## 埃氏筛

从i=2,i*i开始,记录i*j,i*i+1,i*j <=n，记录结果

从i*i开始可以避免重复运算，如果从i * 2开始，会有很多重复

for(int i=2;i*i <= n ; i++)
  for(int j=i;i*j<= n;j++)

如果在数组中，则存在

## 线性筛

存在数比如45，可以同时被 3 5 计算为合数

然后循环到3时，发现45%3==0，跳过，就不用计算5
